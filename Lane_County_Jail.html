Intro

This report aims to provide insight into the current detained population in the Lane County Jail on 2023-03-09. We will use a Python web scrapper to download the data from the jails online inmate search, then format that data into a single table that we will then inspect and generate statistics about the population therein.

Source at https://github.com/russl-corey/Lane_County_Jail

Data

Inmate data was retrieved from the Lane County Jails booking search web portal:

Charges

Measure 11

One of the metrics the sheriff has used is concerning inmates awaiting Measure 11 charges. Measure 11 defines minimum mandatory sentences for listed felonies.

No pre-trial adults in custody being held on Measure 11 charges have been released due to capacity in nearly 10 years (since July 1, 2013).
Sheriff Cliff Harrold to Board of County Commissioners, on Jan 25, 2023

So let us inspect the porportion of inmates that are currently booked on Measure 11 charges.
